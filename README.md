# SuiteCRM-Framework
Full parameterized java testing framework featuring TestNG tests and Cucumber Test in one repository.

## Pre-requisites
- Java version-17
- TestNG 7.6.1
- Cucumber 7.4.0
- Selenium Java 4.4.0
